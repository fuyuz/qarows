# qarows

QA シート（テストケース × 端末/環境）に特化した Web アプリ。

スプレッドシートで QA マトリクスを運用するときの課題——行・列のスクロール、ステータス入力の煩雑さ、絞り込みのしづらさ、自分に関係ない端末/環境まで見えてしまうこと——を、ブラウザ上の専用 UI で解消することを目的としている。

主な利用者は QA 担当（テスト作成・実行）と開発者（バグ対応・修正確認）。UI 言語は日本語メイン（将来 i18n で英語対応を予定）。

## 名称

**qarows** は **QA**（Quality Assurance）と **rows**（行）を組み合わせた名前。テストケースと端末/環境のマトリクス表をイメージしている。

| | |
|---|---|
| 読み（日本語） | **カローズ** |
| 発音記号（IPA） | /ˈkæroʊz/ |

*qa* を「カ」、*rows* を「ローズ」（末尾は *z* 音）と続けて読む。
